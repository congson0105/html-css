Bài 31: element + element, element ~ element

element + element: chọn các phần tử đầu tử đầu tiên ngay sau phần tử, ví dự div + p: chọn phần tử p đầu tiên được đặt ngay sau phần tử div

element ~ element: chọn tât cả các phần tử ngay sau phần tử, ví dụ: div ~ p: chọn tất cả các phần tử của p ngay sau phần tử div
